Extensor de WiFi TL-WA850RE

El TL-WA850RE de TP-LINK está diseñado para expandir cómodamente la cobertura y mejorar la intensidad de la señal de una red inalámbrica existente para eliminar las “zonas muertas”. Con velocidades inalámbricas N a 300Mbps, botón Range Extender y diseño de tamaño en miniatura para montaje en pared, expandir una red inalámbrica nunca había sido tan sencillo. Además, su Puerto Ethernet permite al TL-WA850RE actuar como un adaptador inalámbrico para convertir un dispositivo cableado en uno inalámbrico.

TL-WR820N - Router Wi-Fi Multimodo
    La velocidad de Wi-Fi de 300 Mbps es ideal para las tareas diarias, incluida la navegación, el envío de correos electrónicos y la transmisión de video. Tambien te detallamos otras caracteristicas importantes que posee este router:
  • IPTV admite IGMP Proxy / Snooping, Bridge y Tag VLAN para optimizar la transmisión de IPTV
  • Guest Network proporciona acceso independiente para invitados mientras protege la red del host
  • Los controles parentales administran cuándo y cómo los dispositivos conectados pueden acceder a Internet
  • El firewall SPI y el cifrado inalámbrico protegen su red doméstica
  • Compatible con IPv6 (Protocolo de Internet versión 6)
  • Modo extensor a partir de la V2.
